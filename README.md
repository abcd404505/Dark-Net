# ☠️ DARK NET SMS STRIKER v7.0 ☠️
**High-Performance SMS Injection Tool for Security Research.**

---

## 📌 Description
**DARK NET SMS STRIKER** is a multi-threaded SMS injection engine designed to test API resilience and network security. It utilizes local and global endpoints to simulate high-volume SMS traffic.

## 🚀 Features
* **Multi-API Rotation:** Randomly cycles through Mytel, Wave Money, and Global APIs.
* **Smart Formatting:** Automatically converts local phone formats (09/959).
* **Anonymity:** Built-in support for `proxies.txt` and randomized User-Agents.
* **Optimized UI:** Real-time attack monitoring with a custom console interface.

## 📥 Installation
Clone the repository and install the necessary dependencies:

```bash
pip install -r requirements.txt
```
